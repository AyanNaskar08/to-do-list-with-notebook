# 📝 Daily Planner & To-Do List Web App

A clean, interactive, and user-friendly **Daily Planner & To-Do List** web application designed to help you organize tasks, manage your day, and stay productive. This project combines a planner and notebook-style interface into one seamless experience.

---

## 🚀 Features

* ✅ Add, edit, and delete tasks easily
* 📌 Categorize tasks (All, Pending, Done, Work, Personal, Health, Others)
* 🌙 Dark mode support for better usability
* 🗂️ Notebook-style layout for organized planning
* 🔍 Filter tasks based on status or category
* 📅 Daily planning interface
* 💾 Local storage support (tasks persist after refresh)
* 📊 **Export tasks to Excel format (.xlsx / .csv)**

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling, layout, and dark mode
* **JavaScript (Vanilla JS)** – Functionality and interactivity

---


## 🎯 How It Works

1. Enter your task in the input field
2. Assign a category (optional)
3. Mark tasks as completed when done
4. Use filters to view specific tasks
5. Switch to dark mode for a better experience
6. Export your task list into **Excel format** for external use

---

## 📊 Export to Excel Feature

This application allows users to convert all task data into an **Excel file**, making it easier to:

* 📁 Store records offline
* 📤 Share task lists
* 📈 Analyze productivity

### How it works:

* Tasks are collected from local storage
* Converted into structured format (JSON → CSV/Excel)
* Automatically downloaded as a file
